Abreviação

A tag <abbr> (ou Elemento de Abreviação HTML) represpostaenta uma abreviação e opcionalmente fornece uma
descrição completa do objeto. Se prespostaente, o atributo title="" deve conter a descrição do elemento abreviado,
para observar o efeito, basta posicionar o cursor do mouse no objeto abreviado.
